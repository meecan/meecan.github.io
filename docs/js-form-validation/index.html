Javascript
Form Validation Plugin

A form validation plugin that is completely free to use and easy to implement. It is built using vanilla JavaScript and offers a user-friendly experience.

Download Plugin See On Github

Features

Real-time change validation control.
Disableable first time errors.
Client-side & server-side validation support.
Tested on various browsers.
Highly customizable.
Free and easy to use.
Dependency free.
Like all JavaScript form validation plugins, it is only provide styling. For security reasons, server-side validation should be performed to ensure proper validation.

Installation

Use CDN links below or download now on Github.

<link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/meecan/js-form-validation/validation.css">
<script src="https://cdn.jsdelivr.net/gh/meecan/js-form-validation/validation.js"></script>
const form = document.getElementById('myForm')
window.onload = form.validation()

First Time Errors

To hide errors on initial load, please follow the example provided below.
The default value for the "firstTime" property has been set to "true".

document.querySelector('#myForm').validation({
    firstTime: false
})

Server Side Integration

Your server should return JSON data in the format provided in the second example.

const form = document.getElementById('myForm')

const resultFromAPI = (data) => {
    if (data.status == false) {
        //invalid form
        form.validation.setMessage(data.messages)
    }else{
        //valid form
        form.validation.setMessage({})
        alert('Successfuly...')
    }
}

const requestToAPI = () => {
    const mailValue = form.querySelector('#email').value
    const passwordValue = form.querySelector('#password').value

    const sentData = {
        emailInput: mailValue,
        passwordInput: passwordValue
    }

    fetch('http://example.com/path/to/api', //your API url
        {
            method: 'POST',
            headers: {
                'Content-Type': 'application/json'
            },
            body: JSON.stringify(sentData)
        })
        .then(response => response.json())
        .then(data => {
            resultFromAPI(data) //your form configuration
        })
        .catch(error => console.error(error))
}

window.onload = form.validation({
    realTime: false,
    onSubmit: requestToAPI //your API connection
})
{
    "status": false, //invalid form
    "messages": {
        "inputName": "message example..."
        "inputName2": "message example 2..."
    }
}


{
    "status": true, //valid form
}
